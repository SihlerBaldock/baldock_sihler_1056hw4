# Web Development 1 HW4 - Nested Grid & Flexbox
In this assignment, students are to create a hero size image section for a webpage, ensuring to use responsive image strategies as well as populating the hero section with some text and content. The CSS order property should be used as well to control the visual layout and to change it at different breakpoints. The final result should be fully responsive.

## Installation
No installation required. Simply download the zip file and extract.

## Usage
Open index.html in the browser of your choice.
